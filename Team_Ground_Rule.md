## 1️⃣ 회의 진행

1. 최대한 30분 내로 진행
2. 회의 1시간 전 진행 사항 및 안건 작성
    
    ```java
    [2026. 00. 00]
    
    <진행 사항>
    - {내용}
    
    <진행 예정 사항>
    - {내용}
    ```
    
3. 타 파트가 진행한 사항 확인하고 회의 참석

## 2️⃣ 진행 상황 공유

1. 하나의 토픽에 대해서는 스레드 생성 후 언급
2. 질의는 아래 서식에 맞추어 진행
    
    ```java
    ## {요청 제목}
    ### <업무 요청 내용>
    - {내용을 채워주세요}
    - 담당자 : @ {담당 파트}
    
    ### <업무 요청 배경>
    - {내용을 채워주세요}
    
    ### <업무 요청 데드라인>
    - {2025.00.00 (요일) 오후 00:00}
    ```
